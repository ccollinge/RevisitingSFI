# RevisitingSFI
Reproducible code for analyses used to "revisit" the social frailty index in aging adults as presented in:

Collinge, CW, Bartolomucci, A, & Luciana M. (2026). Revisiting the social frailty index in aging adults: Relevance and implications for an emerging construct. 
Geroscience. https://doi.org/10.1007/s11357-026-02183-1

Find any bugs? Have any questions? Please email collinge@umn.edu
